# Мій HTML-сайт (lab03-html)

## Що реалізовано
- Головна сторінка з семантичною структурою, списками, таблицею, зображенням
- Сторінка "Про мене" з відео та аудіо
- Сторінка "Контакти" з формою зворотного зв'язку та HTML5-валідацією
- Додано просту інтерактивність через JavaScript (кнопка змінює текст)

## Як запустити
1. Відкрийте файл `index.html` у браузері
2. Для зручності можна використовувати Live Server у VS Code

## Структура
- `index.html` — головна сторінка
- `pages/about.html` — про мене
- `pages/contact.html` — контакти
- `assets/img/` — зображення
- `assets/media/` — відео та аудіо
- `app.js` — скрипт для інтерактивності
